# chat-gpt-insert-text-browser-extension

A browser extension that easily inserts a ChatGPT response on editable elements.

## How do I use it?

1. Install the extension
2. Go to an editable element
3. Right click
4. Select `Insert ChatGPT response`
5. Write a text
6. Send it
